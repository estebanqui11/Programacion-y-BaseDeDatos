## 🚀 Instalacion:

1.

 ```bash
    git clone "URL DEL REPOSITORIO"
 ```
2. Abrir Visual Studio Code y situarce en la carpeta del repositorio.

3. Abrir la terminal y ejecutar el siguiente comando:

 ```bash
    npm intall
 ```
4. Iniciar el proyecto

 ```bash
    npm run dev
 ```
